## Multi-Agent Support System

### 1. Report Agent — CUSUM Analysis and Daily Reporting
    Analyzes CUSUM outputs to identify key trends, deviations, and anomalies. Generates a clear, concise daily report summarizing insights and highlighting potential areas of concern.

### 2. Protocol Agent — Emergency Protocol Retrieval
    Retrieves and delivers relevant emergency protocol information to support timely decision-making and effective response actions.
